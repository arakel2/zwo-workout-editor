About Simple ZWO Workout Editor

A simple browser-based editor for creating and editing Zwift workout files (.zwo).

  • Click + Segment to add workout blocks (inserted after the selected segment)
  • Click on a chart block or segment row to select it
  • Use Export .zwo to download your workout for Zwift
  • Use Import .zwo to load and edit existing workouts
  • Power values are in % of FTP

Supported segment types: Warmup, Cooldown, SteadyState, Ramp, IntervalsT, FreeRide, MaxEffort.

Built with vanilla JavaScript. No data is sent anywhere — everything runs locally in your browser.
